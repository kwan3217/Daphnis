# Daphnis
Daphnis, moon of Saturn that clears the Keeler gap and one of the more scenic
spots in the entire Solar System.

# Installation
* Link all the files in these folders individually in your install of Orbiter2024
* Make the following links:
  * `Textures/Deimos`->`Textures/Daphnis`
  * `Textures/Deimos.dds`->`Textures/Daphnis.dds`
  * `Textures/DeimosM.bmp`->`Textures/DaphnisM.bmp`
* If necessary (IE other addons have modified it), add a reference to Daphnis in `Config/Sol.cfg`
```
Saturn:Moon9 = Daphnis
```
